# 🛡️ PolskieLobbyAC - Weryfikator Klienta

Witaj! Ten mod to oficjalny system weryfikacji po stronie klienta dla systemu **PolskieLobbyAC** (Edycja: *Mounts of Mayhem*).

## 🎮 Dla Graczy
Jeśli administrator serwera poprosił Cię o pobranie tego moda, oznacza to, że serwer jest chroniony przez rygorystyczny, autorski system AntyCheat. 

**Instalacja jest banalnie prosta:**
1. Pobierz najnowszą wersję moda (plik `.jar`).
2. Wrzuć plik do swojego folderu `mods` w kliencie Fabric.
3. Gotowe! Wejdź na serwer. 

Mod automatycznie dokona bezpiecznej autoryzacji (Handshake) w tle. Jest niezwykle lekki, nie obciąża komputera, nie zabiera FPS-ów i nie wymaga od Ciebie żadnej dodatkowej konfiguracji.

## 👑 Dla Właścicieli Serwerów
Ważna uwaga: ten projekt to wyłącznie **klient (Mod)**, który komunikuje się z serwerem. Zapewnia on autoryzację, ale sam z siebie nie chroni serwera przed oszustami. 

Aby w pełni chronić swój serwer, odrzucać nieautoryzowane wejścia i skutecznie blokować nieuczciwych graczy (np. niedozwolone modyfikacje ruchowe czy auto-clickery), potrzebujesz dedykowanego, zaszyfrowanego **Pluginu Serwerowego PolskieLobbyAC**.

Projekt wtyczki serwerowej jest aktywnie rozwijany, a sprzedaż i dystrybucja odbywają się prywatnie.

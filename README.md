## 🇬🇧 English Version

Welcome! This mod is the official client-side verification system for the **PolskieLobbyAC** (Edition: *Mounts of Mayhem*).

### 🎮 For Players
If a server administrator asked you to download this mod, it means the server is protected by a strict, custom AntiCheat system.

**Installation is incredibly simple:**
1. Download the latest version of the mod (the `.jar` file).
2. Drop the file into your `mods` folder in the Fabric client.
3. Done! Join the server.

The mod automatically performs a secure Handshake in the background. It is extremely lightweight, does not drain system resources, causes zero FPS drops, and requires absolutely no configuration from you.

### 👑 For Server Owners
**Important note:** This project is strictly the **client (Mod)** that communicates with the server. It provides authorization, but it does not protect the server from cheaters on its own.

To fully protect your server, reject unauthorized connections, and effectively block malicious players (e.g., movement hacks, auto-clickers), you need the dedicated, encrypted **PolskieLobbyAC Server Plugin**.

The server-side project is actively developed, and its distribution is private. Want to secure your server, join the testers, or purchase a license?

Witaj! Ten mod to oficjalny system weryfikacji po stronie klienta dla systemu **PolskieLobbyAC** (Edycja: *Mounts of Mayhem*).

Jeśli administrator serwera poprosił Cię o pobranie tego moda, oznacza to, że serwer jest chroniony przez rygorystyczny, autorski system AntyCheat.

**Instalacja jest banalnie prosta:**
1. Pobierz najnowszą wersję moda (plik `.jar`).

Mod automatycznie dokona bezpiecznej autoryzacji (Handshake) w tle. Jest niezwykle lekki, nie obciąża komputera, nie zabiera FPS-ów i nie wymaga od Ciebie żadnej dodatkowej konfiguracji.
